# Expat.GH 🌍 **News, jobs, real estate and community for Africans living abroad.** A fully functional, production-ready news website for the African diaspora. Built as a clean, multi-page HTML/CSS/JS project — no frameworks, no build tools required. --- ## 🚀 Live Preview Open `index.html` in any browser — no server needed. --- ## 📁 Project Structure ``` expat.gh/ ├── index.html # Homepage ├── css/ │ └── style.css # All styles ├── js/ │ └── main.js # All interactions ├── images/ # Add your images here └── pages/ ├── jobs.html ├── realestate.html ├── visa.html ├── community.html ├── culture.html ├── directory.html ├── newsletter.html ├── about.html ├── advertise.html ├── contact.html ├── write-for-us.html ├── terms.html ├── privacy.html └── cookies.html ``` --- ## ✅ Features - Sticky header with full dropdown navigation (8 nav items, each with subcategories) - Breaking news ticker - Hero article with overlay text - Category tabs (News / Finance / Visa / Business / Expat Life / Culture / Health / Work) - Article list with thumbnails - Real estate property grid - Right sidebar: - Newsletter signup - Trending articles - Community poll - Featured jobs - Browse by country (10 African nations) - Business directory - Article modal popups with full content - Affiliate ad banners (monetization-ready) - Toast notifications - Mobile responsive design - Hamburger menu for mobile - Smooth animations --- ## 💰 Revenue Lanes Built In | Lane | How | |---|---| | Affiliate ads | Swap `.ad-inner` button links with Lemfi, Wise, WorldRemit referral URLs | | Newsletter sponsorships | Connect `#nlEmail` form to Mailchimp or ConvertKit | | Job listings | Charge companies to post in `.job` widget | | Directory listings | Charge businesses to appear in `.dir-item` widget | | Real estate | Partner with African property platforms for referral fees | | Display ads | Replace `.ad-block` sections with Google AdSense or direct sponsors | --- ## 🛠️ How to Deploy ### Option 1 — GitHub Pages (Free, instant) 1.
